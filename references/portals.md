# Job Portals

Default set of job portals for the search. All are free to apply unless marked otherwise. Keep paid portals **off by default** and only enable them if the user explicitly opts in.

## Free — on by default

| Portal | Domain | Notes |
|---|---|---|
| LinkedIn Jobs | linkedin.com/jobs | Broadest reach; good for filtering by remote + region |
| We Work Remotely | weworkremotely.com | Remote-first, strong for tech/dev |
| Remotive | remotive.com | Curated remote roles, many LATAM-friendly |
| NoDesk | nodesk.co | Remote roles, includes agentic/AI category |
| Wellfound (AngelList) | wellfound.com | Startups; equity + remote |
| Working Nomads | workingnomads.com | Aggregator, good coverage |
| Get on Board | getonbrd.com | Strong in LATAM |
| Torre.ai | torre.ai | LATAM-focused, profile-based matching |
| Computrabajo | co.computrabajo.com | Largest job board in LATAM; strong in Colombia |
| elempleo | elempleo.com | Leading job board in Colombia |
| TrabajosYa | trabajosya.com | Trabajando.com group; strong in Colombia, Peru, Ecuador |
| Magneto365 | magneto365.com | Colombia-focused job board |
| Bumeran | bumeran.com | Strong in Argentina, Mexico, Peru, Chile |
| Indeed | indeed.com | Massive global aggregator; free to search and apply |
| Glassdoor | glassdoor.com | Global, includes salary/company reviews alongside listings |
| Upwork | upwork.com | Freelance/contract; good for niche stacks |

## Regional note

Some portals surface roles that require residency or work authorization in a specific country. When a role may not be open to the user's region, flag it rather than silently including it.

## Extending the list

The user can add portals. When they name one, include it in searches using `site:<domain>` queries alongside the keyword terms.
